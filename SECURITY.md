# Security policy

Found a vulnerability in Shield Scanner? Please report it privately via
**GitHub Security Advisories**:

1. Go to the repository's **Security** tab.
2. Click **Report a vulnerability**.
3. Include reproduction steps and the affected version.

Please do **not** open a public Issue, Discussion, or PR for a security bug —
public disclosure before a fix puts every user at risk.

This is a small personal project, so response times are best-effort. False
positives and detection misses are regular bug reports — please open a
public Issue for those.
